# Harriet Boone Woodcraft Website

This project was created for the INFT1016 Information Technology Fundamentals assessment.

## Website Purpose

The website promotes Harriet Boone’s handmade wooden jewellery and accessories business.  
It showcases products, provides background information about the artisan, and allows visitors to contact Harriet.

## Website Pages

The website contains five main pages:

- Home (index.html)
- About (about.html)
- Products (products.html)
- Contact (contact.html)
- Blog (blog.html)

## Technologies Used

The website was built using the following technologies:

- HTML for page structure
- CSS for layout and styling
- JavaScript for interactive features

## Interactive Feature

The contact page includes a JavaScript form validation system that checks whether the user has filled in all fields before submitting a message.
